Create a custom research agent using langchainrb. The agent will take a list of soccer clubs and it will need to research players that have played for the club in the last 20 years.
For each player it should find the following info:
  - When they started playing for the club
  - when they left the club
  - how many appearances they made
  - name of the club
  - name of the player
  - club's league
  - club's country
